# TuPrimeraPaginaGonzalez

## Descripción
Proyecto web en Django con patrón MVT.

## Funcionalidades
- Crear Autor
- Crear Post
- Crear Comentarios
- Buscar Post por título

## Orden para probar

1. / → Página principal
2. /autor/ → Crear autor
3. /post/ → Crear post
4. /comentario/ → Crear comentario
5. /buscar/ → Buscar post

## Tecnologías
- Python
- Django
